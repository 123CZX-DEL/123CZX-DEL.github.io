生成 GitHub Token
快速步骤：
  1. 访问 https://github.com/settings/tokens
  2. 点击 "Generate new token (classic)"
  3. 名称填 "Message Box Token"
  4. Expiration 选 "No expiration"
  5. ✅ 勾选 repo 权限
  6. 点 "Generate token"
  7. 复制生成的 token
粘贴 Token
你的 Token 只会保存在你的浏览器本地，不会上传到服务器。
